# Admin Portal (Vue + Vite)

This is a minimal admin portal scaffold built with Vue 3 and Vite.

## Requirements
- Node.js (16+ recommended)
- npm or pnpm

## Setup (PowerShell)

```powershell
cd f:/DevCode/wk1/admin-portal
npm install
npm run dev
```

Open the URL printed by Vite (usually `http://localhost:5173`).

## What is included
- `vue` + `vue-router` for routes
- `pinia` for simple state (mock users)
- Basic `AdminLayout` with sidebar and topbar
- Views: Dashboard, Users, Settings, Login
